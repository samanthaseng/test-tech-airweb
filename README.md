# Test technique Airweb - Samantha Seng

## Initialisation de l'API
Depuis la racinedu projet, exécuter les commandes suivantes afin d'installer les modules et lancer le serveur :
```bash
npm install
npm start
```

## Initialisation du Front 
Depuis la racine du projet, exécuter les commandes suivantes afin d'installer les modules et lancer l'application :
```bash
cd reactapp/
npm install
npm start
```
